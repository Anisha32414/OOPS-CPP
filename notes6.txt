(2)Inheritance
- When properties and member functions of the base class are passed to the derived class is called inheritance.

- It is used for code reusability.

- A class already have some properties and methods and we want to use them in some other class. So, instead of 
  writing entire code again we simply inherite the properties and methods of the base class in some other class 
  which is the derived class.

Ex. class teacher{                             //base class -> parent class
    public:
       string name:
       string dept;

       teacher(){
          cout<<"Parent class constructor called"<<endl;
       }

       ~teacher(){
          cout<<"parent class destructor is called"<<endl;
       }
    };
    
    class student:public teacher{              //derived class -> child class
    public:
        string subject;

        student(){
            cout<<"student class(derived class) constructor called"<<endl;
        }       

        ~student(){
            cout<<"student class destructor called"<<endl;
        }
    };


Types of Inheritance:-
1. Single Inheritance: one parent having one child
2. Multiple Inheritance: a child having multiple parents
3. Multi-level Inheritance: a child have a parent and that parent is also a child of another parent class
4. hybrid Inheritance: multiple Inheritance are combined together in this type
5. Hierarchial Inheritance: a single parent class is inherited by multiple child classes
