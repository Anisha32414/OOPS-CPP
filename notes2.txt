4 Pillars of OOPS:-


<1> Encapsulation --->

* It is wrapping up of data and methods together in a single unit called CLASS.

Ex. class teacher{
    private:
        int salary;
    public:
        string name;
        string dept;
        int age;

        void update_dept(string department){
            dept=department;
        }

        void set_sal(int sal){
            salary=sal;
        }
        int get_sal(){
            return salary;
        }
    };

    ----------->>>>> above written class "teacher" is an example of ENCAPSULATION.
    ----------->>>>> We use data hiding in Encapsulation for hiding sensitive information
    ----------->>>>> Access Modifiers are used in encalpsulation for data hiding:-
                            * private
                            * public
                            * protected


# Constructor :-

  A special Method invoked everytime when a new object is created.
      ----> Same name as class.
      ----> Doesn't have a return type.
      ----> Only called once(automatically), when object is created.
      ----> Memory allocation happen when constructor is called.  ---->> Memory will be assigned to every object separately when constructor is called.
      ----> constructor is automatically created by the compiler if not created by the programmer.

* In this we can assign the value to some variables automatically during object creation. 

    Ex. class Animal{
        private: 
           int price;
        public:
           string name;
           string sound;

           Animal(){
                cout<<"I am constructor!! New object is created";
                sound="Bark";

                //Sound will be automatically set as "Bark" everytime a new object is created.
           }  
        };

* There are two types of constructors:-
i.  Non-Parameterized -> it do not have any parameter in constructor like given in above example.
ii. Parameterized -> it have parameters in constructor. example given below:-
     
    Ex. class Animal{
        private: 
           int price;
        public:
           string name;
           string sound;

           Animal(string n,string s,int p){
                name=n;
                sound=s;
                price=p;
           }  
        };

## By using parameterized and Non-parameterized constructor in a class is an example of "Constructor Overloading"--->Polymorphism.
